  • Describe what packets are and how they are used to transfer data.

    • Information that has been broken into smaller fragments to be sent, to be later re-pieced together and shown as a whole.

  • Understand the differences between a web page, web server, web browser and search engine.

    • A web page is a file on a servers hard drive.
    A web server is a collectio of web pages which are grouped and usually connected together.
    A web browser is a program that allows you to visit websites.
    A search engine is a web service that helps you find other web pages.

  • Briefly explain what a server is.

    • A computer directly connected to the internet.
